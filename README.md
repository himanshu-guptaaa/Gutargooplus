# Chai Shots Clone

A pixel-perfect React + Vite clone of [chaishots.in](https://chaishots.in/) with theme: **Black & #df4119**.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## 📦 Build for Production

```bash
npm run build
npm run preview
```

## 🎨 Theme

- **Primary**: `#df4119` (Orange-Red)
- **Background**: `#000000` (Pure Black)
- **Fonts**: Bebas Neue (display), Syne (body), Inter (text)

## ✨ Features

- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Sticky navbar with scroll effect
- ✅ Hero section with floating phone mockups
- ✅ Infinite marquee scrollers
- ✅ Animated sections with IntersectionObserver
- ✅ Image carousel (Clap section)
- ✅ Scrolling show thumbnails (portrait + landscape)
- ✅ Interactive moments/tabs slider
- ✅ Animated FAQ accordion
- ✅ App Store & Google Play CTAs
- ✅ Full footer with social links
